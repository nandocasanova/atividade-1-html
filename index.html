História dos computadores

voltar

O primeiro computador foi criado entre 1943 e 1946 e se chamava ENIAC. Cerca de 10 anos depois, foi lançado o RAMAX 305, uma máquina de armazenamento em disco feito pela IBM. No começo, só empresas e certas instituições faziam uso de um computador e, com o passar do tempo, essa tecnologia seria acessada por todos.

O que são Clients?

voltar

Cliente é um termo empregado em computação e representa uma entidade que consome os serviços de uma outra entidade servidora, em geral através do uso de uma rede de computadores numa arquitetura cliente-servidor

O que são Servers?

voltar

Um servidor é um computador equipado com:

  1. um ou mais processadores
  2. bancos de memória
  3. portas de comunicação
  4. aplicativos Softwares/li>
  5. e ocasionalmente, algum sistema para armazenamento de dados, como hard disks internos ou memórias SSD.
O servidor, portanto, centraliza todas as informações e demais dados dentro de uma rede.

Como é uma estrutura básica de um HTML?

voltar

A estrutura básica do HTML é composta das tags !DOCTYPE, head e body. Obviamente, precisamos de mais tags para que a nossa página possua algum conteúdo, porém internamente o HTML é composto e executado por essas três principais tags.

Entendendo a diferença entre listas ordenadas e não ordenadas

voltar

A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados - a lista ordenada enumera os elementos; lista não-ordenada coloca marcadores antes de cada elemento.
